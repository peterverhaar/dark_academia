

The notebook "the Secret History" can be run in Google Colab as folows:

* Open Google Colab via https://colab.research.google.com/
* Click on "File" > "Upload Notebook"
* Click on "Github" and enter the url of this repository: https://github.com/peterverhaar/dark_academia/
* Select the file "the Secret History.ipynb" and Open it
* Click on "Runtime" > "Run All"

If the notebook produces error messages, choose "Runtime" > "Restart session and run all"
